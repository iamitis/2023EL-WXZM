## 功能

- 识别实时音频或语音，屏幕显示生成的文字。
  - 音频语言选择；开始识别/停止识别；清空文字；下载文字。
- 选择或取消翻译模式。
- 翻译模式下，可翻译蓝色显示框内的文字。
  - 目标语言选择；点击翻译；清空文字；下载文字。
- 普通/简洁模式互调。

## tips

音源语言需与选择框里的语言一致。无法选择语言时尝试刷新网页。

建议使用Edge或Chrome浏览器。
